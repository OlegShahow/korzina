


2. Добавление проверки на стороне клиента (перед отправкой формы)
Можно добавить простую валидацию на клиенте перед отправкой данных, чтобы убедиться, что все поля заполнены корректно. Это поможет избежать ошибок на этапе отправки формы.

javascript
Копировать код
document.getElementById("orderForm").addEventListener("submit", function(event) {
    // Останавливаем отправку формы для проверки данных
    event.preventDefault();

    // Получаем значения из скрытых полей
    const order = document.getElementById("orderField").value;
    const ware = document.getElementById("wareField").value;
    const color = document.getElementById("colorField").value;

    // Простой контроль: проверяем, что все поля заполнены
    if (!order || !ware || !color) {
        alert("Пожалуйста, убедитесь, что все данные корректно заполнены.");
        return;
    }

    // Если все данные корректны, отправляем форму
    this.submit();  // Отправка формы
});
Этот код отменяет стандартное действие формы (отправку данных на сервер) до того, как данные будут проверены. Если хотя бы одно поле не заполнено, то будет показано сообщение об ошибке.
